# MusicSharp
![.NET Core](Core) [![Platforms](https://img.shields.io/badge/platforms-desktop-orange.svg)]() [![C#](https://img.shields.io/badge/CSharp-darkgreen.svg)](https://en.wikipedia.org/wiki/C_Sharp_(programming_language)) [![License](https://img.shields.io/badge/License-GPL-blue.svg)](https://www.gnu.org/licenses/gpl-3.0.en.html)

MusicSharp is a Cross-platform CLI based music player written in C#, and has the goal of being minimalistic and light on resources.

Currently in an early alpha beta stage, MusicSharp makes use of the [NAudio](https://github.com/naudio/NAudio) and [and gui.cs](https://github.com/migueldeicaza/gui.cs) libraries.

## Planned Features

- Play audio files
- Support streaming
- Other features coming soon
